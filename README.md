acomics_to_pdf
==============

### Usage

Install venv + requirements:

```
virtualenv venv
. venv/bin/activate
pip install -r requirements.txt
```

Run:
```
. venv/bin/activate
python acomics_to_pdf.py <comic_slug>
```
